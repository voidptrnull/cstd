Detailed Description

Header file for error handling in the CUtils library.

This file defines the CError structure and associated functions used for representing and managing errors within the library. The CError type provides a standardized way to report and handle errors, including error messages, contexts, and error codes.

The file also includes optional error-handling utilities if the CUTILS_COMP_FUNCTIONS macro is defined.

Function pointer type for error handling functions.

This type defines a function pointer for handling errors represented by CError objects. Error handling functions of this type can be used to log, display, or otherwise process errors.
