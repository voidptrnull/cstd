Detailed Description

Header file for all definitions of functions to be used or provided by the user.

This file defines the functions for performing operations which are required for other structures. This provides a standalone, standardized or one way for the definitions.

The file, if enabled during compilation, also enables default implementations of certain functions. To detect their presence, simply use the HAVE_CSTD_DEFAULTS macro.
