Detailed Description

Header file for the CHeap implementation.

This file defines a set of functions to manage a heap-based priority queue. The heap allows for efficient extraction of the minimum element (for min-heap) or maximum element (for max-heap). It provides functionalities like creating a heap, extracting elements, resizing the heap, and freeing the resources. Error handling is done using predefined macros, which indicate success or different failure cases.
