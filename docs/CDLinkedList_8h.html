Detailed Description

Header file for the CString implementation.

This file defines the functions for managing and manipulating a data structure aimed to form a common implementation of a doubly-linked list. The difference between this and the singly-linked list are purely based on their design difference. They can be used interchangeably on a higher level. Best of efforts have been taken care for optimizing the operations for generic data (that is not having any specific type of data in mind).

The header includes error handling through predefined macros for success and failure cases.
